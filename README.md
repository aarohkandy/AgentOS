# agentOS

AI-First Operating System built on Debian

## Build Instructions

```bash
./build.sh
```

## Test Instructions

1. Load the ISO (`agentOS-amd64.hybrid.iso`) into Oracle VM
2. Boot from ISO
3. Test functionality

## Project Structure

- `config/` - Live-build configuration
- `customization/` - Custom themes, AI system, branding
- `scripts/` - Build and utility scripts
- `build.sh` - Main build script
